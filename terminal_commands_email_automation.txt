🔍 Diagnostics & Debugging
Check a specific script for syntax errors without running it
bashpython -m py_compile phase2_orchestrator.py
Check all your pipeline scripts at once
bashpython -m py_compile phase1_contact_resolution.py phase2_orchestrator.py phase2a_alumni_research.py phase2b_email_generation.py config.py

No output = no errors. It only prints something if it finds a problem.

View the last 50 lines of a log or output file

Watch a file update in real time as your pipeline runs (live tail)
bashtail -f alumni_outreach.json
Search for a specific string across all your Python files (e.g. where is ENRICHMENT_DELAY used)
bashgrep -rn "ENRICHMENT_DELAY" .
Search for a string in a specific file only
bashgrep -n "confidence" phase2b_email_generation.py
Search case-insensitively (e.g. find all places "gemini" appears regardless of casing)
bashgrep -rni "gemini" .
Check if a port is in use (e.g. port 8080 if you ever run a local server)

☁️ Google Cloud / APIs
Trigger the first-time OAuth flow — opens browser for Google sign-in, saves token.json
bashpython main.py phase1

This is intentional — your google_auth_helper.py handles auth automatically on first run of any phase that needs Google APIs.

Manually test that your credentials file exists and is readable
bashpython -c "import json; print(json.load(open('credentials.json')).keys())"
Check that token.json exists (means OAuth has already been completed)
bashls -la token.json
Force a fresh OAuth re-authentication by deleting the cached token
bashrm token.json
python main.py phase1
Verify your Google Sheet ID is loading correctly from .env.local
bashpython -c "import config; print(config.GOOGLE_SHEET_ID)"
Verify your Gemini/OpenRouter API key is loading correctly
bashpython -c "import config; print(config.OPENROUTER_API_KEY[:10], '...')"

The [:10] prints only the first 10 characters so your full key is never exposed in terminal history.
